DRENET:: CONCOURS

Comment s'inscrire ?
Pour ORANGE-CI : #144*421#
Pour MTN : *133*200#
Pour MOOV : *155*7*6#
Pour Trésor Pay : *760*2*5#

Si vous êtes un nouvel entrant et que vous n'êtes pas reconnu dans le fichier 2021-2022, prenez attache avec votre établissement pour régulariser votre situation auprès de la DESPS avant d'effectuer votre inscription. En résumé :

Votre préinscription 2022-2023 dépend de deux éléments :
Votre présence dans le fichier national (matricule valide)
Votre inscription et actualisation dans le fichier 2021-2022 (pour les anciens) ou
Votre Décision d'Affectation ou d'Orientation 2021-2022 (pour les nouveaux)
